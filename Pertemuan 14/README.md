# Manajemen User

## index.php
Halaman ini meminta user untuk memasukkan input username dan password. Kemudian akan dilakukan pengecekkan di database terkait akun dan
level yang dimiliki akun tersebut dengan [cek_login.php](cek_login.php)

<img width="800" alt="image" src="https://github.com/user-attachments/assets/f2158857-5ba5-404f-ab53-bb203bd8505a" />

File cek login akan memberikan header(location "...") ke masing-masing halaman. Kemudian user yang sudah terverifikasi akan dialihkan ke 
halaman sesuai dengan level yang dimilikinya.
